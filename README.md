# nxlookup-web

Desktop GUI for [nxlookup](https://github.com/nexxrt/nxlookup) — domain & IP investigation in a native window.

Built with Flask + pywebview.

![screenshot](screenshots/nxlookup-web_QagktcLVnv.png)

## Install & Run

```bash
git clone https://github.com/nexxrt/nxlookup-web.git
cd nxlookup-web
pip install -r requirements.txt
python run.py
```

Pre-built `.exe` on the [Releases](https://github.com/nexxrt/nxlookup-web/releases) page.
