# nxlookup-ui

Desktop GUI for [nxlookup](https://github.com/nexxrt/nxlookup) — domain & IP investigation in a native window.

Built with Flask + pywebview.

![screenshot](screenshots/nxlookup-ui_(2)_TR1h88Tq2k.png)

## Features

- WHOIS, SSL check, HTTP status, DNS records, IP analysis
- Same engine as [nxlookup CLI](https://github.com/nexxrt/nxlookup)
- Dark theme, no browser needed

## Install & Run

```bash
git clone https://github.com/nexxrt/nxlookup-ui.git
cd nxlookup-ui
pip install -r requirements.txt
python run.py
```

Pre-built `.exe` on the [Releases](https://github.com/nexxrt/nxlookup-ui/releases) page.
